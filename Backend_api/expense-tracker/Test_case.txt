Build and run the Docker 

	docker build -t expense-tracker .
	docker run -d -p 8000:8000 expense-tracker


Post command - 
 
curl -X POST http://localhost:8000/expenses \
  -H "Content-Type: application/json" \
  -d '{
        "amount": 25.50,
        "category": "Groceries",
        "description": "Bought vegetables",
        "date": "2025-07-24"
      }'
